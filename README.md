## joginho do tetriz feito em python 
- é um jogo que as peças ficas decendo e o jogador colocas as peças no lugar certo pra não peder 

-[python]('[text](https://www.python.org)')

1 - preparando o ambiente virtual vou fazer com venv 
```bash
    python3 -m venv my env (pra os que usa  o python 3)

    python -m venv my env (pros que tem  o python )
```

2- agora tem que ativar  o ambiente virtual 
```bash
    (linux) source myenv/bin/activate
    (windows) myenv\Scripts\activate
```

obs: - instalar as depedencias e pronto pode usar traquilamente
